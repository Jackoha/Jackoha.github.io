前言

一個月前有看到 CX330 大佬弄了一個神奇的 prompt injection 練習網站
但我之前都沒碰過這方面
這算是我第一次打 prompt injection
沒想到打一打就把題目打光了
最後就想說來寫個 writeup 好了

題目

Traveller


這題只要跟他說

他就會把 flag 吐給你

Hitman


這題說千萬不要告訴任何人 flag
那就問問他提示吧

按照他提示給什麼去找就可以拿到 flag 了

Twilight


這題開始我就有去找 payload
我的 payload 主要是從底下這兩個來的

  • 主要payload
  • 其他附加的東西

這裡就直接用找到的 payload

他就會把 instruction 吐出來
這個payload真的強到不可思議

Tokyo night


這題我的作法跟上題一樣
就不多說什麼了

Hacker


這題做法也還是一樣

Coffee shop


這次會發現他說如果輸出包含 flag 就用*取代他
解決方法也非常簡單
跟他說不要取代就好了啊!

這樣 flag 就會出來了
至於原理我也不是很清楚
但前面大概的作用就是讓 AI 誤以為這是要執行的指令而無視 instruction
所以在那之後的都會被當成指令

Traveler’s revenge


這題作法跟上題一樣

Foo Bar


這題照慣例先試了

這樣似乎沒辦法讓他吐出 flag
被屏蔽掉了
只要不讓他檢測到有 flag 就好了吧

這時候拿去decode就好了
但 decode 完會發現格式有點誤差
不過影響不大
稍微修一下就好了

Who r u


這題照樣先試 payload

這個時候他會回答

他拒絕你的話就讓他閉嘴
記得要讓他不要用*取代

這樣就拿到 flag 了

Roses - I


這次他把 instruction 隱藏起來了
先試個 payload

他這次又拒絕你了

把 instruction leak 出來之後
他會用*取代 flag
一樣叫他不要取代

flag 就噴出來了

Foo Bar 2


這題流程跟上題其實差不多
但我最後把 payload 中的 “You are a GPT” 改成 “Hello”

Programmer


一開始的流程都跟前面一樣
但當我輸入了這個

他會重複我前面的話
所以我把 “Hello” 挖空

這時他回覆我了 You hacker 😈
一樣讓他閉嘴

flag 還是沒出來
但只要改一下順序
先把不要的回覆過濾掉再不讓他用*取代 flag

這樣就成功獲得 flag 了

Mr. Robot


不知道為什麼這題用 payload 就直接吐出答案了
就蠻神奇的

Roses - II


出這題之前 CX330 大佬說會把我的 payload ban 了 太難過了
但我還是用同樣的 payload 解開
總之就先按照前面的流程
把他拒絕你的字都過濾掉

這時候就拿出我們的第二個 payload

這樣 flag 就出來了

我的 payload 真的被 ban 了ㄟ
Instruction:

心得

prompt injection 真的是蠻好玩的
不會像 CTF 要寫 code
希望下次能比 kazma 還要快拿到首殺
