前言

其實這篇在 AIS3 結束之後就該寫了，只是我超偷懶
加上還有 AIS3 junior，所以就一起寫了

DAY -1

這次在 pre-exam 拿到第 49 名 可惡的 AI 大賽
然後這次有 4 個組別可以選

  • A組 : 軟體、網頁及IoT安全
  • B組 : 典範轉移:AI時代的情資運用及防禦
  • C組 : 跨域資訊安全(金融,物流,電信)
  • D組 : 進階資安攻防競技(CTF)

我自己是選了A組，因為聽說要選自己比較擅長的領域，我想說自己 CTF 不強，又不會 AI，C組也不知道在幹嘛，所以就選了 A組
但那個時候的我沒想到，這個組別可以把正常人搞到沒生活，加上今年特別可怕 (如果有聽說今年狀況應該都知道)
不過我那時候並不擔心，因為我覺得 CX330 超強
這裡順便說一下隊友是誰

  • CX330 aka 成大資安社社長
  • dkri3c1 鑑識大師
  • DD pwn神

DAY 0

這天我一大早就起來開始準備去新竹
什麼? 你問我為什麼報到時間在五點，我卻要那麼早起
那當然是去打 mai 阿，竹喵我來了
反正吃完飯之後就跑去新竹，然後跟 ICEDTEA 的去打 mai
五點多之後就跑去交大報到，接著過去宿舍，去跟組員認識一下
雖然我只有一個組員不認識，而且我隊友都是 junior 助教 我明年也要報助教
這天也順便討論說專題要弄什麼，畢竟之前聽說 AIS3 的專題很可怕
這天討論完就把專題方向定在挖 0-day了
不過我自己沒有太多經驗，那個時候感覺蠻可怕的
順帶一題，這天因為不想睡木板，直接睡在地下室

DAY 1

這天早上起來就先去便利商店買份早餐
聽 dk 說這整個禮拜的早餐都要吃便利商店了
接著就馬上衝去會場報到，前面就都是聽一些介紹
接著就是馬老師超硬的 Windows kernel，不過中間有一些滲透相關的東西，所以還算聽得懂
這時候我去看了一下才發現今年怎麼那麼多 kernel 相關的東西，看來是 pwn 仔的勝利
不過我相信認真聽基本上不會有什麼大問題
中午去蹭了交流會的 subway 來吃，不用付錢 好欸
下午有 AI 跟 kernel 的課，但 AI 我是聽不太懂，不過聽某人說共筆都在亂寫
kernel 的話就是在弄出自己的 kernel，不過我全程也只有都在把指令複製貼上而已
晚餐的話…
其實我已經忘了那天吃什麼，好像是學生餐廳?
不過這一整周的經驗下來，我大力推薦小木屋鬆餅，離會場超近，又不會太貴 我連吃了3天
接下來還有破冰遊戲，不過我整場都在水
晚上就是挖洞 挖洞 挖洞 挖洞 挖洞 挖洞 挖洞 挖洞 挖洞 挖洞
挖累了就直接睡覺，直接睡在地下室 (沒錯，第二天了)

DAY 2

真是清新的早晨，今天的我也在地下室醒來了
這天早上的課我其實沒有太大的興趣，所以就等著吃午餐加挖洞
這裡也要順便說一下，午餐給的時間超少，每次都超趕
下午的紅隊滲透的課程蠻有趣的，我自認學到了不少的東西，我也是第一次看到 ysoserial
也是在這節課的時候，CX 去挖了某個 router 的洞還被他找到 command injection
接下來的課就沒什麼印象，kernel 好難 :((((
晚上有一個產業鏈結交流會，反正就是給一些公司宣傳用，也在那邊遇到很多大佬，還順便去要了簽名

這天晚上助教有來檢查各組進度
輪到我們的時候助教看了一下，pwn2ooown 突然超大聲來了一個恭喜
我們挖到了一個超級水洞，直接被助教們 reject 掉，再挖一個 0-day
不過這個時候方向已經蠻明確的，原本還想說挖不到就去做工具或插件
然後今天沒有睡地下室，但是木板床好硬 ᗜ˰ᗜ

因為 CX330 一直在 AIS3 大力推薦 binary ninja 所以我也來幫他宣傳一下 雖然我都沒用

DAY 3

今天的早餐又是便利商店，好ㄟ
上午認真聽的課只有 web 的部分
接下來我就跟蛤蜊蝦跑去吃漢堡王了，但是那裡超遠，不推薦去
結果這個時候 CX 又挖到了一個 command injection，這個時候就剩環境架起來的問題了
下午又又又双叒叕有 kernel 的課，我真的被 kernel 的課給搞死，真的聽不太懂
最後晚上的時候有 trapa CEO 的演講
老實說，這場演講我收穫蠻多的，我也覺得自己其實蠻偷懶的
自己 CTF 退步都歸在因為考證照太忙這點，我覺得自己超級糟糕
所以我自己打算之後要認真打 CTF，至少要練到之後打 HITCON 能有多一點的貢獻
回到宿舍後，就是全組都在架環境，但那時候不知道這噁心的東西會搞我們超久

DAY 4

今天早餐又是便利商店…嗎?
不，我今天直接沒吃就直接過去了 凌晨四點睡真開心
這天有一整個早上的 windows kernel，雖然很難，但我認為蠻有趣的
順便說一下，我那天還跑錯場次 + 找不到教室在哪
急急忙忙的上完後就跑去吃了小木屋鬆餅，我後來還被那邊的店員認出來，是因為我的鬍子嗎?
下午有橘子的課，這場我蠻期待的，我也沒有見過本人
這裡再次大推橘子的課，上起來非常有趣，他也有展示一些很酷的東西，我自己是上的蠻開心的
最後晚上有專題討論，但我們整組基本上都在架環境，架到專題討論時間都結束了都還沒好
結果 dk 在半夜用 firmAE 架起來了，而且他只花了 10 分鐘
然後接下來就剩我把 exploit 弄好，攻擊方式也只是簡單的 command injection bypass blacklist

這裡順便推薦一下 firmAE 非常好用

DAY 5

早上起來隨便買了早餐來吃，就跑過去了，今天要瘋狂趕專題
這天早上就在弄 exploit + debug
弄到差不多下午就把整個 exploit 給弄好了，就是可以簡單打 revshell 的東西
這天一整個超級放鬆，比起前面幾天的挖洞還有架環境好很多
今天吃爆小木屋鬆餅 + 晚餐跑去外面吃好料的
最後晚上的時候 CX 問說要不要看有什麼其他版本的也有同樣的洞
檢查完過後，還有另外兩個版本也有，之後就可以一併報

DAY 6

這天就是聽聽別人專題還有弄簡報
這裡說個小趣事，404 前一天問我要不要測他們的 prompt injection 防禦
然後我把我之前用過的 payload 砸下去，他整個卡住，然後就把 instruction 全部噴出來了
這天晚上想說要不要把 exploit 改的有趣一點
我們就變成把 router 首頁改成 pwned by 每個組員 + 好駭客的圖片還有打 revshell 回來
最後經過助教評價還有各種修改，我們的簡報一整個看起來超級諧咖

DAY 7

這天，我沒有睡覺，到底是為什麼呢???
我們半夜為了升級 CVSS，而且那個時候有想到說前面發 POST request 是不用傳 session 的
所以可以不需要 admin 直接訪問到內部的頁面，經過 CX 逆向確定後，我們成功讓這個洞變成 critical 了!!!
這天報告的時候，CX 的台風真的超好，我在講的時候也只是按照前一天準備的講，沒耍什麼新花樣
DD 在我們報告前還把之前 qemu 架不起來的東西修好，補了整個簡報的完整度
dk 講起來的時候也都蠻有效果的，隊友真的好扛
不過雖然這次我覺得我們這組弄得夠完整，不過我認為還是比不上其他組
今年真的超可怕，TomCat RCE, 10 個 0-day 各種噁心的東西都出來了，所以我不認為有機會拿最佳專題
不過出乎意料的是還真的被我們拿到了，當下喊到我們這組時我蠻驚訝的，但能拿最佳專題真的蠻開心的
再次感謝隊友的各種幫忙跟努力，你們真的超扛
這裡附上最佳專題跟 fumo 的合照

順帶一提，我在那幾天看到 steam 特價了一個好玩遊戲，我睡前都在玩

AIS3 junior

其實這次去 junior 我是帶著一種去玩的心情過去的
但我只能說那裡也是多災多難，不過跟 AIS3 是不同層面的
一個會讓你體會什麼叫累的值得，另外一個是非常不值得的
不過這些也都還好，但不負責任又沒禮貌這點我真的蠻受不了的
事情都過去了，那這些也就算了，其他隊友都還是蠻扛的，而且我自己也有做錯的地方
我在這裡向我組員道歉
最後還有助教們都很用心 最愛你了 SAMMMMMM
pwn2ooown 大哥擔任美食公道伯帶我們去吃好料的，還請一杯飲料，好ㄟ
我自己明年應該會想投 junior 助教，希望 AIS3 官方不要因為我幹的蠢事把我 reject
我去 HITCON 還被 AIS3 的 staff 認出來 超尷尬的
反正 AIS3 的整個行程差不多就醬

結語

AIS3 整體還是蠻充實的，我自己是收穫滿滿
推薦給每個想去爆肝的，這裡一定會滿足你的每個需求
順便說一下， CX 有把我們發現的洞拿去投 CVE，之後有消息在補，我應該會再發一篇 blog
這樣是不是可以說我沒有 my first ctf 但是在 AIS3 拿到 my first cve
話說有拿我 fumo 上台的都拿了最佳專題 這是一種東方神秘力量嗎
